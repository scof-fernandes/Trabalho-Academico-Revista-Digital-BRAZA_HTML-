# Trabalho-Academico-Revista-Digital-BRAZA_HTML-
Projeto de front-end focado em HTML5 semântico e CSS3. Adaptei uma revista impressa para a web usando Flexbox e CSS Grid, garantindo layout responsivo e navegação persistente (sticky). O código demonstra minha autonomia em transformar protótipos estáticos em interfaces funcionais, com foco em UX e organização técnica.
